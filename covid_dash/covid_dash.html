This Covid Dashboard App was built using Flutter and a REST API. When the app is opened, the API sends a json result to the app and it is parsed into a suitable format. After that the app display the relevant informations. There is a list of alphabetically ordered countries with their flags and total cases shown. When you click on any of the country, it's detail page will be shown. If you click on the more info button, extended stats for that country will be displayed.
